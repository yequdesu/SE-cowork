# 🤔Part·1 点名系统需求

   ##1.1 基础需求
   开发一款支持随机和按学号顺序两种模式的课堂点名系统，支持如下功能：
   （1）从Excel文件中导入学生名单（含学号、姓名、专业等基本信息），并将其保存在FREE Online数据库（如[Free MySQL Hosting](https://www.freemysqlhosting.net/)、[db4free](https://www.db4free.net/about.php)、[FreeSQLdatabase](https://www.freesqldatabase.com)）中；
   （2）支持随机点名和按学号顺序点名，被点到的同学显示其姓名和学号。
   （3）利用数据库存储点名信息，并支持积分排名及其结果可视化（柱形图或折线图显示积分最高的若干名同学的积分及其随机点名次数）；
   （4）支持从数据库中导出当前点名的积分详单（含有学号、姓名、专业、随机点名次数、总积分）；
   （5）积分规则如下：
   - 初始积分为0；
   - 每点到一次且到达课堂，则积分+1；
   - 点名回答问题时，能够准确重复所提问的问题，则加0.5分，否则加-1分；准确回答问题，根据情况加0.5-3分；
   - 总积分越高，被随机点到的概率就越低；

   ##1.2 进阶需求（选作）
   自主设计一些新规则，使得该系统更具趣味性。例如，<font color =darkred>**（仅做参考，最好不要与其重复）**</font>：
   （1）被点名两次后拥有一次点名转移权，下次在被点名时可以转给其他同学；
   （2）每次点名触发随机事件，如<font color =darkred>**双倍加分**</font>，<font color =darkred>**疯狂星期四**</font>积分为<font color =darkred>**50**</font>的因数的同学被点名几率增加等。
